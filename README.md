# NexStream 🎬🍿

O **NexStream** é uma plataforma de streaming moderna e responsiva, desenvolvida como parte da trilha de Front-End da Alura. O projeto simula a interface de escolha de perfis e catálogo de vídeos, focando em uma experiência de usuário fluida e visualmente atraente.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as principais tecnologias do desenvolvimento Web moderno:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada, utilizando Flexbox e Grid para responsividade.
- **JavaScript (Vanilla)**: Lógica de interatividade e manipulação do DOM.
- **Git & GitHub**: Controle de versão e deploy via GitHub Pages.

## 🛠️ Funcionalidades

- [x] Tela de seleção de perfil personalizada.
- [x] Catálogo de filmes e séries responsivo.
- [x] Efeitos de hover e transições suaves.
- [x] Navegação entre páginas (Index e Catálogo).
- [x] Favicon personalizado.

## 📐 Desafios Superados

Durante o desenvolvimento, enfrentei desafios reais de deploy, como:
- Resolução de conflitos de **Case Sensitivity** no Git (diferença entre maiúsculas e minúsculas).
- Otimização de **Cache do Navegador** e caminhos relativos de arquivos.
- Ajustes de prioridade de renderização com **Z-Index** e CSS forçado.
